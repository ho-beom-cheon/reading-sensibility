# Codex Task 01: Scaffold application

Goal: create a runnable mobile web/PWA foundation for 문장순간.

Before coding:

- Read `AGENTS.md`.
- Read `docs/design/munjang-sungan-design-v1.0/docs/00_MASTER_DESIGN_v1.0.md` if present.
- If the repository already has a stack, use it.
- If the repository is empty, scaffold a TypeScript React/Next.js-style app with strict typing.

Build:

- Add basic routes: `/`, `/create`, `/library`.
- Add project scripts: `lint`, `typecheck`, `test`.
- Add minimal CI workflow that runs lint/typecheck/test.
- Add a simple responsive mobile shell.

Do not implement OCR, AI, DB, or rendering yet.

Acceptance criteria:

- App runs locally.
- Lint/typecheck/test pass.
- Routes are present.
- CI workflow exists.
