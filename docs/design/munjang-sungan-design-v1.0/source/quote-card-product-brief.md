# 책 문장 감성 카드 서비스 설계 브리프

이 문서는 ChatGPT 또는 다른 설계 도구에 넘겨서 제품 기획, UX 설계, 화면 설계, 기술 설계를 이어가기 위한 정리 문서입니다.

## 한 줄 컨셉

책 페이지를 찍어 마음에 드는 짧은 문구를 선택하고, 사용자의 감상과 독서 순간을 더해 개인 소장용 감성 이미지 카드로 저장하는 서비스입니다.

## 배경

기존 도서관 좌석 지도는 서울 전체 공공도서관 대비 실시간 좌석 데이터가 붙는 도서관 수가 적어 핵심 가치가 약합니다. 반면 책 행사, 독서 기록, 문장 카드 기능은 사용자가 직접 체감하는 가치가 크고, 도서관/책 생활 서비스로 확장하기 쉽습니다.

이 기능은 단순한 quote card maker가 아니라 "내가 책을 읽은 순간을 이미지로 남기는 독서 아카이브"를 목표로 합니다.

## 문제 정의

종이책을 읽다가 좋은 문장을 발견해도 기록 과정이 번거롭습니다.

- 직접 타이핑해야 합니다.
- 사진만 찍으면 나중에 다시 찾기 어렵습니다.
- 문장을 저장해도 감정, 장소, 날짜, 책 맥락이 사라집니다.
- 예쁘게 저장하려면 별도 디자인 앱을 써야 합니다.

## 제품 목표

- 종이책 문장을 빠르게 기록합니다.
- 책 문구 자체보다 사용자의 독서 순간과 감상을 중심에 둡니다.
- 결과물을 개인 소장용 이미지로 예쁘게 저장합니다.
- AI는 무거운 이미지 생성을 직접 담당하지 않고, 분위기 분류와 템플릿 추천에만 가볍게 사용합니다.
- 공개 공유 플랫폼이 아니라 개인 독서 기록 도구로 시작합니다.

## 핵심 사용자 흐름

1. 사용자가 책 페이지를 촬영하거나 이미지를 업로드합니다.
2. OCR로 텍스트를 추출합니다.
3. 사용자가 저장할 짧은 문구를 선택합니다.
4. 책 제목, 작가명, 페이지, 한 줄 감상을 입력합니다.
5. 사용자가 분위기 또는 장르를 선택합니다.
6. AI가 분위기 태그를 분류합니다.
7. 서버가 미리 준비된 템플릿 중 적합한 후보를 추천합니다.
8. 사용자가 템플릿, 비율, 색감, 폰트 느낌을 고릅니다.
9. 이미지가 렌더링되고 PNG/JPG로 저장됩니다.

## MVP 범위

MVP에는 다음 기능만 포함합니다.

- 이미지 업로드
- OCR 텍스트 추출
- 문장 선택
- 책 제목, 작가명, 페이지, 내 감상 입력
- 템플릿 5개 이상
- 이미지 비율 선택: 1:1, 4:5, 9:16
- PNG 다운로드
- 개인 소장용 안내
- 원본 이미지 서버 저장 최소화 또는 미저장

MVP에서 제외합니다.

- 공개 피드
- SNS 자동 공유
- 사용자 간 팔로우
- 긴 문단 저장
- 페이지 전체 카드화
- 책 전체 요약
- 매번 AI 이미지 생성

## 경쟁력 방향

이미 OCR 기반 독서 기록 앱이나 문장 카드 앱은 존재합니다. 따라서 차별점은 기능 개수보다 포지셔닝에 있습니다.

### 1. 문장 카드가 아니라 독서 순간 카드

카드의 중심을 책 문구만으로 두지 않습니다.

포함 요소:

- 선택한 짧은 문구
- 내 감상
- 읽은 날짜
- 읽은 시간
- 책 제목
- 작가명
- 페이지
- 읽은 장소 또는 분위기

예시:

```text
밤 11:42
책을 덮었는데도 이 문장은 계속 남아 있었다.

"선택한 짧은 문구"

『책 제목』 작가명 · p.108
```

### 2. 미리 만든 고품질 템플릿 풀

AI가 매번 디자인을 새로 만들지 않습니다. 사람이 설계한 고품질 템플릿을 많이 준비하고, 각 템플릿에 태그를 붙입니다.

AI는 책/문장/감상의 분위기를 분류하고, 서버는 태그 점수로 템플릿을 추천합니다.

### 3. 한 권의 문장 시리즈

사용자가 같은 책에서 만든 카드를 하나의 시리즈로 묶습니다.

- 이 책에서 남긴 문장
- 이번 달의 문장
- 작가별 문장
- 인생 문장 모음
- 도서관 행사에서 남긴 문장

### 4. 도서관/책 행사와 연결

기존 프로젝트 방향과 연결하려면 다음 흐름이 좋습니다.

- 서울 책/도서관 행사 지도
- 행사 상세에서 읽을 책 또는 작가 저장
- 행사 참여 후 오늘의 문장 카드 생성
- 도서관별 독서 기록
- "이 도서관에서 남긴 문장들"

## AI 사용 전략

AI는 비용이 큰 작업을 하지 않고, 가벼운 분류와 추천만 담당합니다.

AI 없이 처리할 것:

- OCR
- 문장 선택
- 템플릿 렌더링
- 폰트/배경/색상 변경
- 이미지 비율 변경
- PNG/JPG 저장

AI를 사용할 것:

- 문장의 분위기 분류
- 장르/감상 기반 템플릿 추천 태그 생성
- 카드 제목 후보 생성
- 색감/배경 스타일 추천

비싸거나 후순위인 것:

- 책 페이지 이미지를 AI 비전 모델에 그대로 보내기
- OCR 전체 페이지를 AI에 보내기
- 책 전체 줄거리 분석
- 매번 AI 배경 이미지 생성

## 추천 아키텍처

```text
사용자 입력
  - 책 제목
  - 장르
  - 선택 문구
  - 내 감상
  - 원하는 분위기

OCR
  - 이미지에서 텍스트 추출
  - 사용자가 문구 직접 선택

AI 분위기 분류
  - mood
  - genreTone
  - visualKeywords
  - recommendedTemplateTags

템플릿 추천 엔진
  - 템플릿 태그와 AI 태그를 점수화
  - 상위 3개 템플릿 추천

렌더링 엔진
  - 선택한 템플릿에 텍스트와 메타데이터 주입
  - Canvas/SVG/HTML 기반 이미지 생성

다운로드
  - PNG/JPG 저장
```

```text

  + ratio support * 10
```

## 저작권과 안전 설계

이 서비스는 책 문구를 다루므로 처음부터 안전 설계를 넣어야 합니다.

제품 방향:

- 개인 소장용 독서 기록 도구로 시작합니다.
- 공개 공유 플랫폼으로 시작하지 않습니다.
- 문구보다 사용자의 감상과 독서 순간을 중심에 둡니다.

기능 제한:

- 페이지 전체 변환 금지
- 긴 문단 저장 제한
- 여러 페이지 연속 저장 제한
- 책 내용을 복원할 수 있는 대량 저장 제한
- 원본 책 사진 서버 저장 최소화
- OCR 텍스트 장기 저장 선택권 제공
- 공개 공유 기능은 MVP에서 제외

권장 표기:

```text
이 이미지는 개인 독서 기록용으로 생성됩니다.
공개 게시 또는 상업적 이용 시 저작권 문제가 발생할 수 있습니다.
```

출처 필드:

- 책 제목
- 작가명
- 출판사
- 페이지
- 읽은 날짜

## 개인정보와 데이터 저장 원칙

- 원본 책 사진은 기본적으로 서버에 저장하지 않습니다.
- OCR 처리 후 원본 이미지는 삭제합니다.
- 사용자가 저장을 선택한 문구와 메타데이터만 저장합니다.
- 사용자가 언제든 카드와 문구를 삭제할 수 있어야 합니다.
- 공개 공유 기능이 없으면 기본 가시성은 private입니다.

## API 초안

```text
POST /api/ocr
  - image upload
  - returns extracted text blocks

POST /api/theme/classify
  - bookTitle, genre, selectedQuote, reflection, preferredMood
  - returns mood tags and style tags

GET /api/card-templates
  - returns available templates and metadata

POST /api/card-templates/recommend
  - mood tags, genre, text length, ratio
  - returns top templates

POST /api/cards/render
  - selectedQuote, reflection, source, templateId, ratio, renderOptions
  - returns image file or download URL

POST /api/reading-moments
  - saves private reading moment
```

## ChatGPT 설계용 프롬프트

아래 프롬프트를 ChatGPT에 붙여넣고 상세 설계를 요청할 수 있습니다.

```text
나는 책 페이지 사진을 찍어 OCR로 문장을 추출하고, 사용자가 선택한 짧은 문구와 자신의 감상을 바탕으로 개인 소장용 감성 이미지 카드를 만들어주는 서비스를 설계하려고 한다.

중요한 방향:
- 단순 quote card maker가 아니라 "독서 순간을 이미지로 남기는 개인 아카이브"가 목표다.
- AI는 매번 이미지를 생성하지 않는다.
- 미리 만든 고품질 템플릿 풀을 준비하고, 각 템플릿에 mood, genre, layout, textLength, ratio, palette 태그를 붙인다.
- AI는 책 제목, 장르, 선택 문구, 내 감상, 원하는 분위기를 받아 mood/style 태그만 가볍게 분류한다.
- 서버는 템플릿 태그와 AI 태그를 점수화해서 top 3 템플릿을 추천한다.
- 저작권 리스크를 줄이기 위해 공개 피드와 SNS 자동 공유는 MVP에서 제외한다.
- 원본 책 사진은 저장하지 않거나 최소화하고, 개인 소장용 안내를 제공한다.

MVP 기능:
1. 이미지 업로드
2. OCR 텍스트 추출
3. 문장 선택
4. 책 제목, 작가명, 페이지, 내 감상 입력
5. 템플릿 추천
6. 비율 선택: 1:1, 4:5, 9:16
```
