# 문장순간 QA 테스트 계획

## 1. 테스트 환경

| 구분 | 대상 |
|---|---|
| 브라우저 | Chrome, Safari, Samsung Internet |
| 디바이스 | iPhone, Android, desktop Chrome |
| 이미지 | 선명한 책 사진, 흐릿한 사진, 반사 있는 사진, 어두운 사진 |
| 언어 | 한국어 우선, 영어 혼합 케이스 보조 |

## 2. 테스트 시나리오

### 업로드/OCR

- [ ] JPG 업로드 성공
- [ ] PNG 업로드 성공
- [ ] HEIC 업로드 성공 또는 명확한 미지원 메시지
- [ ] 1장 초과 업로드 차단
- [ ] OCR 성공 시 문장 리스트 표시
- [ ] OCR 실패 시 재촬영/직접 입력 표시
- [ ] OCR 세션 만료 후 재업로드 안내

### 문장 선택

- [ ] 문장 1개 선택 가능
- [ ] 선택 문장 수정 가능
- [ ] 120자 초과 시 진행 차단
- [ ] 직접 입력 가능

### 기록 입력

- [ ] 책 제목 필수 검증
- [ ] 작가명 필수 검증
- [ ] 감상 필수 검증
- [ ] 페이지 선택 입력 가능
- [ ] 분위기/장르 선택 가능

### 추천/렌더링

- [ ] AI 성공 시 추천 top 3 표시
- [ ] AI 실패 시 fallback 추천 표시
- [ ] 1:1 PNG 생성
- [ ] 4:5 PNG 생성
- [ ] 9:16 PNG 생성
- [ ] 긴 문구에서 overflow 없음
- [ ] 다운로드 URL 만료 처리

### 보관함

- [ ] 로그인 사용자 저장 가능
- [ ] 비로그인 사용자 저장 시 로그인 안내
- [ ] 목록 조회 가능
- [ ] 상세 조회 가능
- [ ] 삭제 가능

### 보안/개인정보

- [ ] 원본 이미지 저장 안 함 확인
- [ ] OCR 전체 텍스트 TTL 삭제 확인
- [ ] 타 사용자 ReadingMoment 접근 차단
- [ ] 생성 카드 private 접근 제어

## 3. 릴리스 차단 기준

- [ ] 정상 입력에서 카드 생성이 불가능하다.
- [ ] 원본 이미지가 의도치 않게 저장된다.
- [ ] 타 사용자 카드에 접근 가능하다.
- [ ] 공개 공유 기능이 노출된다.
- [ ] 120자 초과 문구가 저장된다.
- [ ] PNG 다운로드가 동작하지 않는다.
