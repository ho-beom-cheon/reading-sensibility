# 문장순간 v1.0 - 릴리스/핸드오프 설계서

## 1. 목적

v0.1부터 v0.9까지의 설계를 개발 착수와 MVP 릴리스에 넘길 수 있는 상태로 고정한다.

## 2. v1.0 릴리스 범위

### 포함

- 모바일 웹/PWA 기반 카드 생성 플로우
- 책 페이지 이미지 업로드
- OCR 문장 후보 추출
- 문장 선택/수정
- 책 정보/감상 입력
- AI 분위기 분류 및 fallback
- 템플릿 5종 추천
- 1:1, 4:5, 9:16 PNG 렌더링
- PNG 다운로드
- 로그인 사용자 private 보관함 저장
- 원본 이미지 미저장
- OCR 세션 TTL
- 삭제 기능

### 제외

- 공개 피드
- SNS 자동 공유
- 팔로우
- 긴 문단 저장
- 페이지 전체 카드화
- 여러 페이지 연속 OCR
- 책 전체 요약
- AI 이미지 생성

## 3. 개발 착수 조건

| 항목 | 완료 기준 |
|---|---|
| PRD | MVP 포함/제외 범위 확정 |
| IA/화면 | 화면 ID와 필드/검증 확정 |
| 데이터 | DB schema 초안 작성 |
| API | OpenAPI 초안 작성 |
| AI | prompt/schema/fallback 작성 |
| 템플릿 | MVP 5종 메타데이터 작성 |
| 렌더링 | 해상도/overflow 정책 확정 |
| 저작권/개인정보 | 저장 정책/고지 문구 확정 |
| QA | 테스트 계획과 이벤트 정의 |

## 4. 릴리스 통과 기준

| 영역 | 통과 기준 |
|---|---|
| OCR | 선명한 한국어 책 페이지 기준 성공률 80% 이상 |
| 카드 생성 | 정상 입력 기준 생성 완료율 95% 이상 |
| 렌더링 | 3개 비율에서 clipping/overflow 없음 |
| 다운로드 | PNG 파일 정상 다운로드 |
| 보관함 | 저장/조회/삭제 가능 |
| 보안 | 타 사용자 리소스 접근 차단 |
| 개인정보 | 원본 이미지 저장 안 함 검증 |
| 저작권 안전 | 공개 피드/SNS 자동 공유 없음 |

## 5. 운영 모니터링

출시 후 2주 동안 다음 지표를 매일 확인한다.

- OCR 실패율
- OCR 수정률
- 문장 선택 완료율
- 기록 입력 완료율
- 카드 렌더링 실패율
- PNG 다운로드율
- 보관함 저장률
- flow_abandoned step 분포

## 7. 최종 결론

`문장순간` v1.0 설계는 “책 원문 공유 서비스”가 아니라 “개인 독서 순간 아카이브”를 기준으로 한다. MVP는 OCR, 문장 선택, 감상 입력, 템플릿 추천, PNG 저장에 집중하며, 커뮤니티/공유/대량 저장 기능은 제외한다.
