# 문장순간 v0.3 - 화면/와이어프레임 명세

## 1. 목적

화면별 UI 컴포넌트, 입력값, 버튼, 검증 조건, 에러 메시지, API 연동을 개발 가능한 수준으로 정의한다.

## 2. 화면 목록

| 화면 ID | 화면명 | 목적 |
|---|---|---|
| HOME | 홈 | 서비스 이해 및 시작 |
| CREATE_UPLOAD | 업로드 | 책 페이지 촬영/업로드 |
| OCR_PROCESSING | OCR 처리 | 이미지 분석 상태 표시 |
| SELECT_QUOTE | 문장 선택 | OCR 결과에서 문구 선택/수정 |
| MOMENT_DETAILS | 기록 입력 | 책 정보와 감상 입력 |
| TEMPLATE_SELECT | 템플릿 선택 | 추천 템플릿/비율 선택 |
| PREVIEW | 미리보기 | 최종 카드 확인/렌더링 |
| SAVE_COMPLETE | 저장 완료 | PNG 다운로드/보관함 저장 |
| LIBRARY | 내 보관함 | 생성 카드 목록 |
| MOMENT_DETAIL | 상세 보기 | 문장/감상/카드 상세 확인 |

## 3. HOME

### 목적

사용자가 “책 페이지를 찍으면 독서 순간 카드가 된다”는 가치를 즉시 이해하게 한다.

### 주요 UI

- Hero copy
- 예시 카드 2~3개
- `책 페이지 찍기` CTA
- `이미지 업로드` CTA
- `내 보관함` 링크
- 원본 이미지 미저장 안내

### 화면 문구

```text
책에서 만난 한 문장을, 오늘의 감상으로 남기세요.

사진으로 찍고,
문장을 고르고,
그때의 마음을 카드로 남깁니다.
```

### 이동

| 버튼 | 이동 |
|---|---|
| 책 페이지 찍기 | CREATE_UPLOAD camera mode |
| 이미지 업로드 | CREATE_UPLOAD album mode |
| 내 보관함 | LIBRARY |

## 4. CREATE_UPLOAD

### 주요 UI

- 업로드 드롭존
- 카메라 버튼
- 앨범 선택 버튼
- 직접 입력하기 버튼
- 촬영 가이드
- 개인정보/원본 미저장 안내

### 검증

| 조건 | 메시지 | 처리 |
|---|---|---|
| 파일 없음 | 이미지를 선택해 주세요. | 진행 차단 |
| 파일 크기 초과 | 이미지 용량이 너무 큽니다. | 재선택 |
| 지원 형식 아님 | JPG, PNG, HEIC 이미지만 사용할 수 있습니다. | 재선택 |
| 해상도 낮음 | 글자가 흐릿할 수 있습니다. 다시 촬영을 권장합니다. | 경고 후 진행 가능 |

### API

`POST /api/ocr`

## 5. OCR_PROCESSING

### 상태

| 상태 | 설명 | UI |
|---|---|---|
| uploading | 이미지 전송 중 | progress |
| processing | OCR 처리 중 | spinner |
| success | OCR 성공 | 자동 이동 |
| failed | OCR 실패 | 재촬영/직접 입력 |

### 문구

```text
문장을 읽는 중입니다.
인식된 문장은 다음 단계에서 직접 수정할 수 있습니다.
```

## 6. SELECT_QUOTE

### 주요 UI

- OCR 문장 후보 리스트
- 선택 radio
- 선택 문구 편집 textarea
- 글자 수 카운터
- 긴 문장 경고
- 직접 입력 모드

### 입력 제한

| 항목 | 제한 |
|---|---:|
| 선택 문구 | 최대 100자 권장, 120자 하드 제한 |
| 선택 문구 개수 | 1개 |
| OCR 전체 저장 | 금지 |

### 문장 길이 분류

| 길이 | 분류 | 처리 |
|---:|---|---|
| 1~40자 | short | 모든 템플릿 가능 |
| 41~90자 | medium | 일반 템플릿 가능 |
| 91~120자 | long | 일부 템플릿만 가능 |
| 121자 이상 | blocked | 다음 단계 차단 |

## 7. MOMENT_DETAILS

### 필드

| 필드 | 필수 | 타입 | 제한 | 기본값 |
|---|---:|---|---:|---|
| bookTitle | Y | string | 80자 | 없음 |
| author | Y | string | 50자 | 없음 |
| publisher | N | string | 50자 | 없음 |
| pageNumber | N | string | 20자 | 없음 |
| reflection | Y | string | 150자 | 없음 |
| mood | N | enum | - | 미선택 |
| genre | N | enum | - | 미선택 |
| readAt | Y | datetime | - | 현재 시각 |
| placeLabel | N | string | 30자 | 없음 |

### 검증 메시지

| 조건 | 메시지 |
|---|---|
| 제목 없음 | 책 제목을 입력해 주세요. |
| 작가 없음 | 작가명을 입력해 주세요. |
| 감상 없음 | 이 문장을 만난 감상을 적어 주세요. |
| 감상 초과 | 감상은 150자 이내로 입력해 주세요. |

## 8. TEMPLATE_SELECT

### 주요 UI

- 추천 템플릿 top 3
- 전체 템플릿 보기
- 비율 선택: 1:1, 4:5, 9:16
- 강조 방식: 문장 중심, 감상 중심, 날짜/출처 중심
- 추천 이유 요약

### API

1. `POST /api/theme/classify`
2. `POST /api/card-templates/recommend`
3. `GET /api/card-templates`

## 9. PREVIEW

### 주요 UI

- 카드 미리보기
- 비율/템플릿 변경 버튼
- 출처 표시 토글
- PNG 생성 버튼
- 개인 소장용 안내

### API

`POST /api/cards/render`

## 10. SAVE_COMPLETE

### 주요 UI

- PNG 다운로드 버튼
- 내 보관함에 저장 버튼
- 새 카드 만들기 버튼
- 내 보관함 보기 버튼

### 정책

- 다운로드 URL은 1시간 만료한다.
- 보관함 저장 시 로그인 필요.
- 저장되는 데이터는 선택 문구, 감상, 책 정보, 렌더링 설정이다.

## 11. LIBRARY

### 주요 UI

- 카드 목록
- 책별 필터
- 날짜별 필터
- 검색
- 삭제

## 12. MOMENT_DETAIL

### 주요 UI

- 생성 카드
- 선택 문구
- 내 감상
- 책 제목/작가/페이지
- readAt/placeLabel
- 삭제 버튼
