# 문장순간 v0.5 - API/오류 계약

## 1. 목적

프론트엔드, 백엔드, OCR, AI, 렌더링 엔진 간 계약을 고정한다.

## 2. API 목록

| Method | Path | 목적 |
|---|---|---|
| POST | /api/ocr | 책 페이지 이미지 OCR 처리 |
| POST | /api/theme/classify | 선택 문구/감상 기반 분위기 분류 |
| GET | /api/card-templates | 사용 가능한 템플릿 조회 |
| POST | /api/card-templates/recommend | 템플릿 top 3 추천 |
| POST | /api/cards/render | 카드 PNG 렌더링 |
| GET | /api/cards/{cardId}/download | PNG 다운로드 |
| POST | /api/reading-moments | 독서 순간 저장 |
| GET | /api/reading-moments | 보관함 목록 조회 |
| GET | /api/reading-moments/{id} | 독서 순간 상세 조회 |
| DELETE | /api/reading-moments/{id} | 독서 순간 삭제 |

## 4. 핵심 API 상세

### 4.1 POST /api/ocr

책 페이지 이미지를 업로드하고 OCR 문장 후보를 반환한다.

- Request: multipart/form-data
- Response: ocrSessionId, expiresAt, sentences
- 저장 정책: 원본 이미지는 처리 후 삭제한다.

### 4.2 POST /api/theme/classify

사용자 선택 문구와 감상으로 mood/style 태그를 생성한다.

- AI 입력에는 OCR 전체 텍스트를 보내지 않는다.
- selectedQuote는 최대 120자다.
- reflection은 최대 150자다.

### 4.3 POST /api/card-templates/recommend

AI 태그와 사용자가 선택한 비율/강조 방식으로 템플릿 top 3을 반환한다.

### 4.4 POST /api/cards/render

선택된 템플릿에 문장, 감상, 출처 정보를 주입해 PNG를 생성한다.

## 5. 오류 코드 체계

| 코드 | HTTP | 설명 | 사용자 메시지 |
|---|---:|---|---|
| INVALID_IMAGE | 400 | 지원하지 않는 이미지 | 사용할 수 없는 이미지입니다. |
| IMAGE_TOO_LARGE | 413 | 이미지 용량 초과 | 이미지 용량이 너무 큽니다. |
| OCR_FAILED | 422 | OCR 실패 | 문장을 충분히 인식하지 못했습니다. |
| QUOTE_TOO_LONG | 422 | 선택 문구 길이 초과 | 120자 이내로 줄여 주세요. |
| REQUIRED_FIELD_MISSING | 400 | 필수값 누락 | 필수 정보를 입력해 주세요. |
| AI_CLASSIFY_FAILED | 502 | AI 분류 실패 | 기본 추천으로 보여드립니다. |
| TEMPLATE_NOT_FOUND | 404 | 템플릿 없음 | 템플릿을 찾을 수 없습니다. |
| RENDER_FAILED | 500 | 렌더링 실패 | 카드를 생성하지 못했습니다. |
| DOWNLOAD_EXPIRED | 410 | 다운로드 만료 | 다운로드 링크가 만료되었습니다. |
| UNAUTHORIZED | 401 | 로그인 필요 | 보관함 저장은 로그인이 필요합니다. |
