# 문장순간 v0.2 - 사용자 플로우/IA

## 1. 목적

사용자가 서비스를 처음 접한 순간부터 PNG 저장 또는 보관함 저장까지 이동하는 전체 경로를 고정한다.

## 2. 핵심 사용자 플로우

```text
홈
  → 책 페이지 업로드
  → OCR 처리
  → 문장 선택/수정
  → 책 정보·감상 입력
  → 템플릿 추천/선택
  → 카드 미리보기
  → PNG 다운로드
  → 선택 시 내 보관함 저장
```

## 3. 사용자 경로 상세

| 단계 | 화면 ID | 사용자 행동 | 시스템 처리 | 다음 단계 |
|---:|---|---|---|---|
| 1 | HOME | 카드 만들기 시작 | 세션 초기화 | CREATE_UPLOAD |
| 2 | CREATE_UPLOAD | 이미지 촬영/업로드 | 파일 검증 | OCR_PROCESSING |
| 3 | OCR_PROCESSING | 대기 | OCR 실행, 원본 삭제 | SELECT_QUOTE |
| 4 | SELECT_QUOTE | 문장 선택/수정 | 길이 검증 | MOMENT_DETAILS |
| 5 | MOMENT_DETAILS | 책 정보/감상 입력 | 필수값 검증 | TEMPLATE_SELECT |
| 6 | TEMPLATE_SELECT | 템플릿/비율 선택 | AI 태그 및 추천 계산 | PREVIEW |
| 7 | PREVIEW | 카드 확인 | 렌더링/PNG 생성 | SAVE_COMPLETE |
| 8 | SAVE_COMPLETE | 다운로드/저장 | 파일 제공/DB 저장 | LIBRARY 또는 HOME |

## 5. 주요 사용자 유형별 경로

### 5.1 비로그인 사용자

```text
홈 → 업로드 → OCR → 문장 선택 → 기록 입력 → 템플릿 → 미리보기 → PNG 다운로드
```

- 보관함 저장을 누르면 로그인 안내를 표시한다.
- 다운로드 자체는 로그인 없이 허용한다.

### 5.2 로그인 사용자

```text
홈 → 업로드 → OCR → 문장 선택 → 기록 입력 → 템플릿 → 미리보기 → PNG 다운로드 → 내 보관함 저장
```

- 생성한 ReadingMoment와 GeneratedCard를 private 상태로 저장한다.

### 5.3 OCR 실패 사용자

```text
홈 → 업로드 → OCR 실패 → 재촬영 또는 직접 입력 → 기록 입력 → 템플릿 → 미리보기
```

## 6. 네비게이션 규칙

| 상황 | 뒤로가기 처리 |
|---|---|
| 업로드 전 | 홈으로 이동 |
| OCR 처리 중 | 취소 확인 표시 |
| 문장 선택 후 | OCR 결과 유지 |
| 기록 입력 후 | 입력값 유지 |
| 템플릿 선택 후 | 선택 템플릿 유지 |
| 저장 완료 후 | 홈 또는 보관함으로 이동 |
