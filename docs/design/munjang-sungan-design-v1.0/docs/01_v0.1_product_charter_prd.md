# 문장순간 v0.1 - 제품 정의/PRD

## 1. 목적

`문장순간`은 종이책에서 발견한 짧은 문장을 OCR로 가져오고, 사용자의 감상, 책 정보, 읽은 시간과 함께 개인 소장용 이미지 카드로 저장하는 독서 순간 아카이브다.

## 2. 문제 정의

종이책 독자는 좋은 문장을 발견했을 때 다음 문제를 겪는다.

| 문제 | 설명 | 제품 대응 |
|---|---|---|
| 타이핑 부담 | 문장을 직접 입력하기 번거롭다. | 책 페이지 촬영/OCR 제공 |
| 맥락 소실 | 사진만 저장하면 날짜, 감상, 책 정보가 사라진다. | ReadingMoment 모델로 보관 |
| 검색/회상 어려움 | 사진첩에 책 사진이 섞여 다시 찾기 어렵다. | 책별/날짜별 보관함 |
| 디자인 부담 | 예쁘게 저장하려면 별도 앱이 필요하다. | 고품질 카드 템플릿 제공 |

## 3. 제품 한 줄 정의

> 종이책에서 발견한 한 문장을 사용자의 감상, 시간, 책 정보와 함께 이미지 카드로 보관하는 개인 독서 순간 아카이브.

## 4. MVP 목표

1. 사용자가 책 페이지 이미지를 업로드한다.
2. OCR 결과에서 짧은 문구를 선택하거나 수정한다.
3. 책 제목, 작가, 페이지, 감상을 입력한다.
4. 분위기/장르를 선택한다.
5. 추천 템플릿으로 이미지를 생성한다.
6. PNG 파일로 다운로드한다.
7. 선택적으로 개인 보관함에 저장한다.

## 5. MVP 포함 기능

| 기능 | 우선순위 | 완료 기준 |
|---|---:|---|
| 이미지 업로드 | P0 | JPG/PNG/HEIC 1장 업로드 가능 |
| OCR 텍스트 추출 | P0 | 문장 후보 리스트 반환 |
| 문장 선택/수정 | P0 | 1개 문구 선택 및 직접 수정 가능 |
| 책 정보 입력 | P0 | 제목/작가/페이지 입력 가능 |
| 감상 입력 | P0 | 사용자 감상 1개 저장 가능 |
| 템플릿 추천 | P1 | top 3 추천 표시 |
| 카드 미리보기 | P0 | 선택 템플릿으로 미리보기 표시 |
| PNG 다운로드 | P0 | 1080 기준 PNG 생성 |
| 개인 소장 안내 | P0 | 저장 전 고지 표시 |
| 원본 이미지 미저장 | P0 | OCR 처리 후 삭제 |

|---|---|---:|
| OCR 성공률 | 업로드 후 문장 후보 반환 비율 | 80% 이상 |
| 문장 선택 완료율 | OCR 결과에서 문장 선택 완료 | 70% 이상 |
| 기록 입력 완료율 | 책 정보/감상 입력 완료 | 60% 이상 |
| 카드 생성 완료율 | 미리보기 생성 완료 | 50% 이상 |
| PNG 다운로드율 | 생성 후 다운로드 완료 | 40% 이상 |
