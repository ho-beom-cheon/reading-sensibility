# 문장순간 v0.8 - 보안/개인정보/저작권 안전장치

## 1. 목적

책 문구와 책 페이지 이미지를 다루는 서비스 특성상 MVP부터 저장 범위, 공개 범위, 삭제 정책, 사용자 고지를 명확히 한다.

## 2. 기본 원칙

| 원칙 | 설명 |
|---|---|
| 개인 소장 중심 | MVP는 공개 플랫폼이 아니라 private 기록 도구다. |
| 최소 저장 | 원본 이미지와 OCR 전체 원문을 장기 저장하지 않는다. |
| 사용자 통제 | 선택 문구, 감상, 카드 삭제가 가능해야 한다. |
| 출처 명시 | 책 제목, 작가명, 출판사, 페이지를 입력할 수 있어야 한다. |
| 기능 제한 | 긴 문단, 페이지 전체 카드화, 대량 저장을 제한한다. |

## 3. 저장 정책

| 데이터 | 저장 여부 | 기간 | 비고 |
|---|---:|---|---|
| 원본 책 이미지 | 저장 안 함 | OCR 후 즉시 삭제 | 디버깅 저장도 기본 금지 |
| OCR 전체 텍스트 | 영구 저장 안 함 | 세션 TTL 1시간 | 선택 문구만 저장 가능 |
| 선택 문구 | 저장 가능 | 사용자 삭제 전까지 | 최대 120자 |
| 사용자 감상 | 저장 가능 | 사용자 삭제 전까지 | 최대 150자 |
| 책 정보 | 저장 가능 | 사용자 삭제 전까지 | 출처 필드 |
| 생성 카드 | 선택 저장 | 사용자 삭제 전까지 | private |
| 다운로드 URL | 임시 | 1시간 | 토큰 만료 |

## 4. 저작권 리스크 감소 기능

| 리스크 | 대응 |
|---|---|
| 책 내용 재현 | 선택 문구 120자 제한 |
| 페이지 전체 복제 | 페이지 전체 카드화 금지 |
| 대량 저장 | MVP에서 여러 페이지 연속 OCR 제외 |
| 공개 확산 | 공개 피드/SNS 자동 공유 제외 |
| 출처 누락 | 책 제목/작가/페이지 필드 제공 |
| 서비스 저장 리스크 | 원본 이미지와 OCR 전체 텍스트 미저장 |

## 5. 필수 안내 문구

### 업로드 화면

```text
원본 책 사진은 OCR 처리 후 저장하지 않습니다.
선택한 문구와 직접 입력한 기록만 저장됩니다.
```

### 문장 길이 초과

```text
책 내용을 길게 재현하지 않기 위해 짧은 문구만 저장할 수 있습니다.
120자 이내로 줄여 주세요.
```

### 저장 화면

```text
이 이미지는 개인 독서 기록용으로 생성됩니다.
공개 게시 또는 상업적 이용 시 저작권 문제가 발생할 수 있습니다.
```

## 6. 접근 제어

| 리소스 | 접근 정책 |
|---|---|
| ReadingMoment | owner only |
| GeneratedCard | owner only |
| downloadUrl | token + expiry |
| OcrSession | session owner + TTL |
| CardTemplate | public read |

## 7. 삭제 정책

- 사용자는 ReadingMoment를 삭제할 수 있다.
- ReadingMoment 삭제 시 연결된 GeneratedCard도 soft delete 또는 cascade delete한다.
- 원본 이미지는 저장하지 않으므로 별도 삭제 대상이 아니다.
- 임시 OCR 세션은 TTL 만료 후 자동 삭제한다.
