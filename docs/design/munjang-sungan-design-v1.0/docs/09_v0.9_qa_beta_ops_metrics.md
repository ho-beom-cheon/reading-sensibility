# 문장순간 v0.9 - QA/베타/운영 지표

## 1. 목적

MVP 출시 전 품질 기준, 테스트 범위, 운영 지표, 베타 피드백 수집 방법을 확정한다.

## 2. QA 범위

| 영역 | 테스트 대상 |
|---|---|
| 업로드 | 파일 형식, 용량, 해상도, 실패 처리 |
| OCR | 성공/실패/낮은 confidence/문장 분리 |
| 문장 선택 | 선택, 수정, 길이 제한, 직접 입력 |
| 기록 입력 | 필수값, 글자 수, 날짜/페이지 입력 |
| 추천 | AI 성공, AI 실패, 후보 0개, 비율 필터 |
| 렌더링 | 1:1, 4:5, 9:16, overflow, 다운로드 |
| 보관함 | 저장, 목록, 상세, 삭제 |
| 개인정보 | 원본 미저장, OCR TTL, private 접근 제어 |

## 3. 핵심 테스트 시나리오

| ID | 시나리오 | 기대 결과 |
|---|---|---|
| QA-001 | 선명한 책 페이지 업로드 | OCR 문장 후보 표시 |
| QA-002 | 흐릿한 이미지 업로드 | 경고 또는 OCR 실패 fallback |
| QA-003 | 120자 초과 문구 입력 | 다음 단계 차단 |
| QA-004 | 감상 미입력 | 다음 단계 차단 |
| QA-005 | AI API 실패 | 기본 추천 표시 |
| QA-006 | 9:16 비율 선택 | 1080x1920 PNG 생성 |
| QA-007 | 텍스트 overflow | fontScale 조정 또는 템플릿 부적합 |
| QA-008 | 비로그인 저장 | 로그인 안내 표시 |
| QA-009 | 다운로드 링크 만료 | 만료 메시지 표시 |
| QA-010 | 삭제 요청 | ReadingMoment/GeneratedCard 삭제 |

## 4. 이벤트 지표

| 이벤트 | 발생 시점 | 주요 속성 |
|---|---|---|
| upload_started | 업로드 시작 | sourceType |
| ocr_completed | OCR 완료 | sentenceCount, avgConfidence |
| quote_selected | 문장 선택 | quoteLength, sourceType |
| details_completed | 기록 입력 완료 | hasPage, hasMood, genre |
| template_recommended | 추천 완료 | templateIds, aiConfidence |
| template_selected | 템플릿 선택 | templateId, ratio |
| card_rendered | 렌더링 완료 | templateId, ratio, durationMs |
| card_downloaded | PNG 다운로드 | cardId |
| moment_saved | 보관함 저장 | userId, bookId |
| flow_abandoned | 플로우 이탈 | step |

## 5. 베타 운영 기준

| 항목 | 기준 |
|---|---|
| 베타 대상 | 내부 사용자 10~30명 |
| 테스트 기간 | 1~2주 |
| 수집 항목 | OCR 품질, 감상 입력 부담, 카드 저장 욕구, 템플릿 선호 |
| 핵심 질문 | “생성된 카드를 실제 저장하고 싶은가?” |
| 중단 기준 | OCR 실패율 40% 이상 또는 렌더링 실패율 5% 이상 |
