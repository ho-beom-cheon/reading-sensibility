# 문장순간 v0.7 - 렌더링/템플릿/에셋 명세

## 1. 목적

선택 문구, 감상, 책 정보를 고품질 PNG 카드로 렌더링하기 위한 템플릿 시스템을 정의한다.

## 2. 렌더링 파이프라인

```text
ReadingMoment 입력
  + CardTemplate
  + RenderOptions
  ↓
HTML/CSS 또는 SVG 렌더링
  ↓
Headless browser screenshot 또는 SVG rasterize
  ↓
PNG 생성
  ↓
다운로드 URL 발급
```

## 3. 출력 규격

| 비율 | 용도 | 해상도 |
|---|---|---:|
| 1:1 | 저장용 카드 | 1080 × 1080 |
| 4:5 | 피드형 이미지 | 1080 × 1350 |
| 9:16 | 잠금화면/스토리형 | 1080 × 1920 |

## 4. 카드 구성 요소

| 요소 | 필수 | 설명 |
|---|---:|---|
| selectedQuote | Y | 선택한 짧은 문구 |
| reflection | Y | 사용자 감상 |
| bookTitle | Y | 책 제목 |
| author | Y | 작가명 |
| pageNumber | N | 페이지 |
| readAt | Y | 읽은 날짜/시간 |
| placeLabel | N | 장소 |
| mood | N | 분위기 |

## 5. MVP 템플릿 5종

| ID | 이름 | 분위기 | 비율 |
|---|---|---|---|
| quiet_margin_01 | 여백의 문장 | 차분함, 위로, 사색 | 1:1, 4:5, 9:16 |
| rainy_window_01 | 비 오는 창가 | 쓸쓸함, 회상, 위로 | 1:1, 4:5, 9:16 |
| night_desk_01 | 밤의 책상 | 고독, 몰입, 사색 | 1:1, 4:5, 9:16 |
| library_card_01 | 대출카드 | 기록성, 아카이브 | 1:1, 4:5 |
| bookmark_01 | 책갈피 | 간직함, 위로, 사색 | 9:16 |

## 6. 텍스트 오버플로우 정책

| 상황 | 처리 |
|---|---|
| selectedQuote 120자 초과 | 렌더링 차단 |
| reflection 150자 초과 | 입력 단계 차단 |
| 카드 영역 초과 | fontScale 자동 감소 |
| fontScale 0.85 미만 필요 | 템플릿 부적합 처리 |
| 템플릿 부적합 | 다른 템플릿 추천 |

## 7. 렌더링 옵션

```ts
type RenderOptions = {
  emphasis: 'quote' | 'reflection' | 'date' | 'source';
  showPublisher: boolean;
  showPageNumber: boolean;
  showReadAt: boolean;
  fontScale: number;
};
```

## 8. 에셋 관리

| 에셋 | 저장 위치 | 버전 관리 |
|---|---|---|
| 템플릿 배경 | object storage 또는 static assets | templateId 기준 |
| 썸네일 | static assets | templateId + ratio |
| 생성 카드 | user generated storage | cardId 기준 |
| 렌더 설정 | DB JSON | GeneratedCard.renderOptions |
