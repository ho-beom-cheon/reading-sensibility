# 문장순간 v0.4 - 데이터 모델/상태 모델

## 1. 목적

제품의 핵심 엔티티와 생성 플로우 상태를 명확히 정의한다.

## 2. 핵심 엔티티

```text
User 1 ─ N ReadingMoment
Book 1 ─ N ReadingMoment
ReadingMoment 1 ─ N GeneratedCard
CardTemplate 1 ─ N GeneratedCard
OcrSession 1 ─ 1 CreateCardFlow temporary
```

## 9. 생성 플로우 상태

```ts
type CreateCardStep =
  | 'upload'
  | 'ocr'
  | 'select_quote'
  | 'details'
  | 'template'
  | 'preview'
  | 'complete';
```

## 10. 상태 전이

```text
upload
  → ocr
  → select_quote
  → details
  → template
  → preview
  → complete
```

### 예외 전이

| 현재 상태 | 조건 | 이동 |
|---|---|---|
| ocr | OCR 실패 | select_quote manual mode |
| select_quote | 직접 입력 | details |
| template | AI 실패 | template fallback 추천 |
| preview | 렌더링 실패 | template 또는 preview 재시도 |
| complete | 보관함 저장 | library |
