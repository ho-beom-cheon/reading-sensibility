# 오류 코드 명세

| 코드 | HTTP | 원인 | 사용자 메시지 | 복구 방법 |
|---|---:|---|---|---|
| INVALID_IMAGE | 400 | 지원하지 않는 이미지 형식 | 사용할 수 없는 이미지입니다. | 다른 이미지 선택 |
| IMAGE_TOO_LARGE | 413 | 파일 용량 초과 | 이미지 용량이 너무 큽니다. | 압축 또는 재촬영 |
| IMAGE_TOO_BLURRY | 422 | 글자 인식 어려움 | 글자가 흐릿할 수 있습니다. | 재촬영 권장 |
| OCR_FAILED | 422 | OCR 결과 없음 | 문장을 충분히 인식하지 못했습니다. | 재촬영/직접 입력 |
| OCR_SESSION_EXPIRED | 410 | OCR 세션 만료 | 인식 결과가 만료되었습니다. | 다시 업로드 |
| QUOTE_REQUIRED | 400 | 문장 미선택 | 저장할 문장을 선택해 주세요. | 문장 선택 |
| QUOTE_TOO_LONG | 422 | 120자 초과 | 120자 이내로 줄여 주세요. | 문장 수정 |
| REFLECTION_REQUIRED | 400 | 감상 없음 | 이 문장을 만난 감상을 적어 주세요. | 감상 입력 |
| REQUIRED_FIELD_MISSING | 400 | 필수 필드 누락 | 필수 정보를 입력해 주세요. | 필드 보완 |
| AI_CLASSIFY_FAILED | 502 | AI 호출 실패 | 기본 추천으로 보여드립니다. | fallback 추천 |
| TEMPLATE_NOT_FOUND | 404 | 템플릿 없음 | 템플릿을 찾을 수 없습니다. | 다른 템플릿 선택 |
| TEMPLATE_UNSUPPORTED_RATIO | 422 | 비율 미지원 | 이 템플릿은 선택한 비율을 지원하지 않습니다. | 비율 변경 |
| RENDER_FAILED | 500 | 이미지 생성 실패 | 카드를 생성하지 못했습니다. | 재시도 |
| DOWNLOAD_EXPIRED | 410 | 다운로드 토큰 만료 | 다운로드 링크가 만료되었습니다. | 다시 생성 |
| UNAUTHORIZED | 401 | 로그인 필요 | 보관함 저장은 로그인이 필요합니다. | 로그인 |
| FORBIDDEN | 403 | 타 사용자 리소스 접근 | 접근할 수 없습니다. | 요청 중단 |
